# Quiz Create

テキストから一問一答（早押し）または4択クイズを作成するスキル。

## 使い方

1. クイズ形式を確認する（早押し or 4択。指定がなければ早押し）
2. 入力に応じて素材テキストを用意する:
   - **テーマが渡された場合** → `research` スキルで調査して素材テキストを生成する
   - **URLが渡された場合** → `web_fetch` でページ内容を取得して素材テキストとする
3. `{baseDir}/steps.md` を読んで手順に従いクイズを作成する

## 出力形式

**早押しクイズ:**
```
問題文: ～でしょう？
答え: ○○
```

**4択クイズ:**
```
問題文: ～でしょう？
① ...  ② ...  ③ ...  ④ ...
正解: ②
解説: ...
```

## 参照ファイル

- `{baseDir}/steps.md` — 詳細な作成手順
- `{baseDir}/policies.md` — 作問方針・品質基準
